CodeSkulptor link - https://py3.codeskulptor.org/#user307_3jVl1WVFNE_1.py

background.py - contains the Background class which creates a moving background

sprite.py - contains Sprite, Keyboard, and Clock classes which are used for the avatar

explosion.py - contains Explosion_spritesheet class which is used for the game over animation when the avatar 
has no lives left and makes contact with a laser

laser.py - contains Laser_spritesheet and Clock_laser classes which are responsible for the generation of 
the lasers and the logic which checks if the avatar is colliding with the lasers.

interaction.py - contains Interaction class which is used to allow for all the objects made from all the classes
in the project to be able to interact with each other.
